INTERSTATE COMPACT ON THE PLACEMENT OF CHILDREN     

  REFERENCE GUIDE

 

PART I:  General Instructions:

 

This is a reference guide for completing an ICPC 100A or 100B in Georgia SHINES.  Cooperation from the county departments in adhering to the following steps will aid the ICPC staff in expediting results and/or replies for services on behalf of children across state lines.

 

  • All required documents for any ICPC request must be uploaded into External Documentation with the exception of IV-E Eligibility Documentation and the Foster Care Case Plan [FCCP].  If applicable, to the case, this would include a Home Study of the Placement Resource, ICWA Enclosures and/or Proof of Paternity.

 

  • Each uploaded ICPC document must be designated as such on the External Documentation Detail page by selecting the ICPC Document checkbox.  Additional identifying information should be added to the Detail comments box, when applicable. These actions should be done prior to selecting the Complete? button on the ICPC Detail page.

 

  • The Cover Letter should be the final document to be uploaded and checked as an ICPC Document.  This should occur once it is signed by the County Approving Authority. The signature indicates that all documents for the 100A or 100B are ready for processing. 

 

  • Once the letter is signed, the County Approving Authority is responsible for sending an e-mail to the ICPC administrator advising that the ICPC Detail page for a 100A or 100B, and all associated documents, are ready for review. The e-mail should include identifying information such as the Case ID and the child’s name.  In addition, a manual Case Alert to the ICPC Administrator may also be created in Georgia SHINES from the Case To Do List page.

 

  • Upon receipt of the ICPC notification e-mail from the county, the ICPC Administrator will review the ICPC Detail page. The ICPC Administrator will have the ability to edit the page, if necessary.

 

  • The ICPC Administrator will review, print and collate all the 100A or 100B ICPC documents in External Documentation that will be sent to the other state.  The ICPC Administrator will also access the current, approved Georgia SHINES FCCP and IV-E Eligibility Summary page to print and include in a 100A packet.

 

  • If any required document which should be uploaded into External Documentation has not been uploaded and/or designated as an ICPC document, the ICPC Administrator will not process the ICPC request from the county. Exception: In some instances, such as for an Expedited ICPC request, the 100A packet must be sent before certain documents are available.  However, those documents must be uploaded immediately upon receipt.

 

  • To notify the ICPC Administrator that a document has been received and uploaded or that there is an update in case information, the ICPC Administrator should be notified via e-mail.  In addition, a manual Case Alert may be added in Georgia SHINES from the Case To Do List page.

 

  • All calls to the ICPC Administrator should be from the supervisor, unless there is an emergency situation.  Please do not refer parents or relatives to the ICPC staff.

 

  • FL Cases only –A CCFA that is less than 6 months old must be provided or else a current, signed and dated social summary must be submitted in narrative form on DHS_DFCS agency letterhead.  The social summary should include the following: reason the child[ren] entered care, permanency plan, current situation, and a brief child history, including medical, psychological and educational/developmental information.

 

 

 

PART II:  Instructions for ICPC Detail Page - Form 100A

 

Purpose:

The Interstate Compact on the Placement of Children (ICPC) Request form is for use by Social Services staff for children in Foster Care.  The form is used to initiate a request to place a child in DFCS custody with a relative, parent, or foster/adoptive resource in another state.  An ICPC Detail page should be completed for each child in the request.

 

Completion of Page:

A.    Child Identifying Data Section

1.      The following information will auto-populate based on data that is recorded in the child’s FCC stage: First Name/Middle/Last, DOB [date of birth], Gender, SSN [social security number], Race, Ethnicity, Mother: [name] and Father: [name].  If the information is incorrect or incomplete, please update the child’s Person Detail page.

2.      The Title IV-E Determination field will display ‘Yes’ if the child is IV-E eligible and ‘No’ if not eligible, based on the Actual Eligibility as displayed on the most recent Eligibility Summary page. It will display ‘Pending’ if a determination has not been completed.   

 

B.     Form 100A Section

ICWA Eligible: Select a radio button to indicate whether or not the child is a party to the Indian Child Welfare Act.

 

C.     Placement Information Section

1.      Name of Person(s) with whom child is to be placed: Select the name of the Primary Person proposed as a placement resource from the first drop down list.  If applicable, select the name of the Other Caretaker in the Home from the second drop down list. Only principals [PRNs] from the Person List who are 18 and older will appear in the drop down lists.

2.      The following fields will auto-populate based on data that is recorded on the Person Detail page for the Primary Person:

a.       Address:  A complete address is needed to ensure proper routing of the ICPC request.  An address with a P.O. Box number may not be used as a primary address.   The address must be a physical location.

b.      Phone:  A phone number is needed to assist the case worker in the other state with making contact with the proposed resource.

c.       The first SSN – this information is optional.

d.      Relationship

e.       Side of Family: [maternal/paternal] Side of Family is required information, if a Relative Study is requested. 

3.      If an Other Caretaker in the Home is selected, the second SSN will auto-populate based on data that is recorded on his/her Person Detail page. This information is optional.

           

D.    Services Requested Section

1.      Initial Report Requested: Select a radio button to indicate the type of home study requested. 

a.       If a relative foster home study is needed, select Foster Home Study.

b.      Adoptive Home Study: If this option is selected, Voluntary Surrender papers and/or a court order for TPR must be uploaded into External Documentation.

·         Adoption: This field will auto-populate based on the determination from the AA Funding Summary page. This is a required field when an Adoptive Home Study is requested.

·         To Be Finalized In: If Adoptive Home Study is requested, if known, also select a radio button to indicate where the adoption will be finalized. 

 

E.     Documents Enclosed Section

1.      Select all required documents which have been uploaded into External Documentation and have a check in the ICPC Document checkbox to indicate that they are ready to be processed. Remember: If a required document is not yet available [e.g. for some Expedited ICPC requests], it must be uploaded into External Documentation as soon as it is and the ICPC Administrator notified.

2.      Select Case Plan if there is a current APRV FCCP.  If it has been less than 30 days since the child’s removal and an APRV FCCP is not available, notify the ICPC Administrator as soon as it is.

3.      Select IV-E Determination, if one has been made.

4.      Proof of Paternity may be a document, such as a marriage certificate, DNA test results or a court order specifying the paternity of the child, and is required for a:

a.       Relative Home Study request when the Side of Family is paternal and the father is not a legal [putative] father: If proof of paternity is not uploaded into External Documentation, a Foster Home Study will be requested.

b.      Parent Home Study request for a non-legal [putative] father:  If proof of paternity is not uploaded into External Documentation, an ICPC request cannot be made.

 

F.      Case Manager Signature

1.      Case Manager’s Name: When the Complete? button is selected, this field will display the name of the person who has completed the ICPC Detail page.

2.      Date: This field will display the date that the Complete? button is selected on the ICPC Detail page.

 

G.    Date Last Updated: This field will display the last date the ICPC Detail page was modified and saved.

 

Additional Fields on Form 100A:

A.    The following data will auto-populate directly onto the 100A form based on data that is recorded in Georgia SHINES:

1.      To: This field will display <State of the Primary Person> ICPC  

2.      From: This field will always display ‘GA ICPC’. 

3.      Name of the Agency or Person Legally and Financially Responsible for Planning for Child; Phone; and Address. This information will auto-populate based on the legal county of the child.

4.      Type of Care Requested: The system will pre-select the appropriate checkbox to indicate the type of care the proposed resource will provide [Foster Family Home, Relative (Not Parent) or Parent], based on the Initial Report Requested radio button which was selected on the ICPC Detail page.

5.      Current Legal Status of Child: This field will always display ‘Sending Agency Custody/Guardianship’.  It will also display ‘Parental Rights Terminated – Right to Place for Adoption’, if there is a legal status of Permanent Court, Permanent Voluntary or Joint Commitment with DJJ – Permanent Court.

6.      Supervisory Services Requested:  This field will always display ‘Request Receiving State to Arrange Supervision’.

7.      Supervisory Reports Requested: This field will always display ‘Quarterly’ and ‘Other: Monthly Contact Dates’.

8.      Signature of Sending Agency or Person: The name of the person who completed the ICPC Detail page will display.

9.      Date: This field will auto-populate the date the ICPC Detail page was completed.

 

B.     The following fields will remain blank on the Form 100A:

1.       Name and Address of Supervising Agency in Receiving State

2.      Signature of Sending State Compact Administrator, Deputy or Alternate: The ICPC Administrator will print the form, then sign and date it.

 

* At this time for Court Jurisdiction Only and Relative Care Subsidy [RCS] cases, the process will continue to be one that is external from Georgia SHINES.  However, all associated documents for RCS cases should be uploaded into External Documentation from the PFC stage.

 

 

 

PART III:  Instructions for ICPC Detail Page - Form 100B

 

Purpose:

The Interstate Compact on the Placement of Children (ICPC) Report on Child’s Placement Status form is for use by Service Staff for children in Foster Care.  The form is used to document the date of placement of a child in DFCS custody with a relative, parent, or foster/adoptive resource in another state.  The form is also used to withdraw or terminate a placement. 

 

Completion of Page:

A.    Child Identifying Data: The following information will auto-populate based on data that is recorded in the child’s FCC stage: First Name/Middle/Last, DOB [date of birth], Mother: [name] and Father: [name].

 

B.     Name of the Resource

1.      Select the name of the primary person of the placement resource from the drop down list.

2.      The Address and Phone information will auto-populate based on data that is recorded on the Person Detail page for the selected resource.

3.      Type of Care: Select the radio button for the type of placement approved by ICPC.

 

C.    Placement Status

1.      Initial Placement of Child in Receiving State:

a.       If applicable, select the radio button.

b.      Date Child Placed in Receiving State: If the Initial Placement radio button is selected, enter the date the child was or will be placed with the ICPC approved resource.

2.      Placement Change:  

a.       If applicable, select the radio button.

b.      Effective Date of Change: If the Placement Change radio button is selected, enter the date the child was or will be placed with the new ICPC approved resource.

 

D.    Compact Placement Termination

1.      Select the appropriate radio button to indicate the reason for the compact placement termination. 

a.       If the reason is Adoption Finalized:

·         Select a radio button to indicate where it was finalized.

·         Select the checkbox for Court Order Attached.  The court order must be uploaded into External Documentation and the checkbox for ICPC Document selected.

·         If the reason is either Legal Custody Returned to Parent[s] or Legal Custody Given to Relative, select the corresponding checkbox for Court Order Attached The court order must be uploaded into External Documentation and the checkbox for ICPC Document selected.

b.      If Other is selected, the reason must be recorded in the associated text field.

2.      Date of Termination: Enter the date the ICPC case is being closed.

3.      NOTE:  An ICPC case has to be approved for closure by the ICPC office.  Once approved, the county office will receive a copy of the ICPC transmittal indicating the ICPC case is closed.  This transmittal should be uploaded into External Documentation.

 

H.    Case Manager Signature

1.      Case Manager’s Name: When the Complete? button is selected, this field will display the name of the person who has completed the ICPC Detail page.

2.      Date: This field will display the date that the Complete? button is selected on the ICPC Detail page.

 

I.       Date Last Updated: This field will display the last date the ICPC Detail page was modified and saved.

 

Additional Fields on Form 100B:

A.    The following data will auto-populate directly onto the 100B form based on data that is recorded in Georgia SHINES:

1.      To: This field will display <State of the Primary Person> ICPC 

2.      From: This field will always display ‘GA ICPC’. 

3.      The following fields will auto-populate based on the resource who has been selected on the ICPC Detail page:

a.        Name and Relationship - when Legal Custody to Relative is selected as a compact termination reason.

b.      Name of Resource - when Proposed Placement Request Withdrawn is selected as a compact termination reason.

c.       Name of Approved Placement - when Approved Resource Will Not Be Used for Placement is selected as a compact termination reason.

4.      Person/Agency Supplying Information: The name of the person who completed the ICPC Detail page will display.

10.  Date: This field will auto-populate the date the ICPC Detail page was completed.

 

B.     Compact Administrator, Deputy or Alternate: The ICPC Administrator will print the form, then sign and date it.
