Status Description

Statuses have been added to test harness in an effort to increase visibility to the progress of a document.

Converting - The document template or service is undergoing a conversion fix. This means the document should not be unit tested.
Ready to Test - The document is ready to be unit tested using the test scripts.
SIR - The document has a bug and has been logged in the SIR DB tool.
Regression Test - The SIR has been fixed and the document is ready to be tested.
Completed - The document has successfully passed unit test.
